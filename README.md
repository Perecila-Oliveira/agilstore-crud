# agilstore-crud
Sistema de gerenciamento de produtos (CRUD) desenvolvido em JavaScript com Node.js para execução via terminal.

Como Executar o Projeto?

Pré-requisitos: Node.js precisa estar instalado

Passo a Passo

-> Clone o repositório:

git clone <https://github.com/Perecila-Oliveira/agilstore-crud>

-> Acesse a pasta do projeto:

cd agilstore-crud

-> executar a aplicação via terminal 

node index.js
